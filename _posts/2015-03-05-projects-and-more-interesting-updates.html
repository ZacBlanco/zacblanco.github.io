So I've been pretty busy these last two weeks with schoolwork and other matters so I haven't had much time to update anything here.

I did want to let you know that I started working on a Java library use and to implement statistics functions that I think would be useful to analyze large datasets.

For now I'm calling the project DStat (unless someone can come up with a better name). It will hopefully become a fully fledged statistics library for performing operations on multiple types of distributions of data. As of now it only has a few functions, but feel free to take a look at that and the other projects over at my GitHub page. I try to make progress whenever I find free time to do a little extra work. Feel free to contribute too! Everything I do I release as open source.

I eventually would like to be able to take some data you might find from your own Google Data Archive and within a few minutes (using DStat of course!) come up with some neat conclusions based on your own search habits or other usage statistics. Of course it might be some time before that's possible I hope that in time you will be able to find that kind of functionality with it.
